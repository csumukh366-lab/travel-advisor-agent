# PlanMyTrip AI: Intelligent Goal-Based Travel Planning Agent

An advanced travel planning engine that treats itinerary generation as a pathfinding problem. By utilizing A* search and utility-based heuristic functions, this agent optimizes travel routes, budget utilization, and user preferences.

## 🚀 Features
- **Goal-Based Agent:** Uses A* search to find the optimal path in a multi-state environment.
- **Utility-Based Scoring:** Heuristics calculate interest matches, budget compatibility, and route efficiency.
- **Dynamic Logistics:** Real-time flight and hotel allocation based on selected service tiers (Economy/Business/Luxury).
- **Native PDF Export:** Generate professional, formatted travel blueprints directly from the UI.

## 🛠️ Tech Stack
- **Frontend:** Tailwind CSS, JavaScript (Vanilla)
- **Logic:** A* Search Algorithm, Heuristic Scoring
- **Mapping:** Leaflet.js
- **PDF Generation:** Browser-native Print Engine

## ⚙️ Installation
1. Clone the repository: `git clone https://github.com/yourusername/planmytrip-ai.git`
2. Open `index.html` in your preferred web browser.

## 📜 Academic Info (AAT1)
- **Agent Type:** Goal-Based Agent
- **Search Algorithm:** A* (f(n) = g(n) + h(n))
- **Environment:** Partially Observable, Sequential, Multi-State
